##Word Scramble Game

A simple C-based word scramble game where the player must guess the original word from its scrambled version.

#Features
- Random word selection
- Scrambled letters using a shuffle algorithm
- 3 chances to guess the correct word
- Shows the correct answer when attempts run out

#How to Play
 1. Run the program.
 2. A scrambled word appears.
 3. Type your guess and press Enter.
 4. You have 3 attempts to guess correctly.

#Rules
-  You must type the word in lowercase (no spaces).
-  Each wrong guess reduces your remaining chances.
- No hints are provided—rely on your observation skills.
-  Once all 3 chances are used, the game ends.


#Controls
 - Type your guess → Keyboard
 - Submit guess → Enter
